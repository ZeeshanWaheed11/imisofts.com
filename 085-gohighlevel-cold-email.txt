## GHL for Cold Email Agencies: Workflow Example

### Lead Intake (Automated)

1. Contact lands on landing page (built in GHL)
2. Auto-creates contact card
3. Assigns to sales rep based on territory
4. Triggers SMS: "Thanks for interest. Scheduling call."
5. Starts 3-day automated follow-up sequence

### Campaign Management

1. Connect Instantly account to GHL
2. When prospect opens email → GHL updates activity log
3. When prospect clicks → GHL notes in deal
4. When prospect replies → GHL auto-logs conversation
5. Custom pipeline stages (Prospect → Qualified → Call Scheduled → Won)

### Pipeline Tracking

GHL dashboard shows:
- # of leads in each stage
- Deal value (if you assign)
- Expected close date
- Win probability
- Custom metrics (e.g., "emails sent," "reply rate")

### Client Delivery (White-Label)

Sell GHL as "Your Agency's CRM":
- Log in with client credentials
- Client sees only their leads
- You manage backend (integrations, automations)
- Client portal is white-labeled

## Setting Up GHL for Cold Email

### Step 1: Create Account

1. Go to GoHighLevel.com
2. Sign up (email, password, company name)
3. Choose plan (Pro recommended for agencies)
4. Verify email

### Step 2: Create Landing Page

GHL has built-in landing page builder:
1. Dashboard → Websites → New Page
2. Select template (Lead Capture form)
3. Add form fields: First Name, Email, Phone, Company
4. Configure thank you message
5. Publish page (GHL provides unique URL)

### Step 3: Connect Instantly

1. Settings → Integrations → Zapier (or custom webhook)
2. Create Zapier automation:
   - Trigger: Email opened in Instantly
   - Action: Update contact in GHL
3. Log opening time in GHL activity log

**Effect:** When prospect opens cold email, GHL immediately knows (enables dynamic follow-up).

### Step 4: Create Pipeline

1. Dashboard → Pipeline (or Deals)
2. Create custom pipeline:
   - Stage 1: New Lead (just added to list)
   - Stage 2: Engaged (opened email, clicked link)
   - Stage 3: Qualified (replied or scheduled call)
   - Stage 4: Call Scheduled
   - Stage 5: Won (became customer)
3. Set expected value per deal (€2,000 avg)
4. Save

### Step 5: Email Sequences

GHL has built-in email sequences (though most use Instantly instead):

1. Dashboard → Automations → Email Sequences
2. Create sequence:
   - Email 1 (Day 1): Initial cold email
   - Email 2 (Day 4): Follow-up if no response
   - Email 3 (Day 8): Final soft pitch
3. Set trigger: "When added to list"
4. Save

**Note:** Most agencies use Instantly for cold email, then use GHL to track responses + follow-ups. Not just GHL for sending.

### Step 6: Dashboard Customization

Create custom dashboard showing:
- Leads added (daily)
- Reply rate (%)
- Call conversion rate
- Pipeline value (€)
- Sales rep performance

## Common Setup Mistakes

1. **Not Connecting Instantly Properly**
   - Use Zapier webhook to sync Instantly opens/clicks
   - Without this, GHL blind to email performance

2. **Pipeline Too Complex**
   - 5-7 stages max. More = confusion.
   - Keep it simple: New → Engaged → Qualified → Won

3. **Not Using White-Label**
   - If selling to clients, white-label. Otherwise looks cheap.
   - GHL allows full rebranding.

4. **Manual Data Entry**
   - Automate everything. Email opens, clicks, replies all auto-log.
   - Don't make reps manually update GHL.

5. **No Dashboard Monitoring**
   - Create custom dashboard with key metrics
   - Track: leads/day, reply rate, call conversion
   - Review weekly

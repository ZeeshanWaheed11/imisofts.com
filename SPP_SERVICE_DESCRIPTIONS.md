## USAGE INSTRUCTIONS

### For SPP.co Setup:

1. **Navigate to:** Billing → Services → Add Service
2. **Copy the Title** into the "Service Name" field
3. **Copy the Description** into the "Description" field
4. **Set the Price** as indicated in each service
5. **Choose Billing Type:**
   - Infrastructure/Campaign packages: One-time
   - Digital Marketing: Recurring monthly
   - Development services: One-time/Project
6. **Save** and move to next service

### Pro Tips:
- Enable "Featured" tag for your most popular services
- Add service images/icons if available
- Set up package deals for bundled services
- Create coupon codes for first-time clients
- Add testimonials to service descriptions later

### Order Form Setup:
Group services into logical order forms:
1. **Email Infrastructure** - All 3 infrastructure packages
2. **Campaign Setup** - All 3 campaign packages
3. **Development Services** - CRM, Shopify, Web Development
4. **Marketing Services** - Digital Marketing, AI Automation
